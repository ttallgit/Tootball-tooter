## Tootball Tooter

A retro-styled penalty shootout game. A flashing arrow tells you where the goalkeeper will dive and you must shoot in the **opposite** direction to score.

### Controls

| Key | Action |
|---|---|
| `↑` | Run forward |
| `←` `→` `↓` | Choose shot direction |
| `SPACE` | Shoot |
| Direction + `SPACE` | Execute the shot (press within the timing window) |

### Game Modes

#### Normal
Arrow shows LEFT or RIGHT. Shoot the opposite way. Timing tolerance: **30ms**.

#### Hard
Arrow can also point UP — goalkeeper stays center, shoot DOWN (↓) to score. Timing tolerance: **15ms**.

### How to Play

1. Pick a mode and a tiger
2. Press ↑ to start your run
3. A flashing arrow shows where the goalkeeper will dive
4. Press the **opposite** direction + SPACE to shoot

| Arrow | Shoot |
|---|---|
| ← | → + SPACE |
| → | ← + SPACE |
| ↑ (Hard only) | ↓ + SPACE |

5 shots per round. Score all 5 for a perfect game.

### Playable Tigers

Roary (orange), Atlas (blue), Sege (green), Nessie (purple), Telix (red) — cosmetic only.
